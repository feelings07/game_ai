# Game AI

2D top-down RPG на Godot 4.7 (GDScript). Процедурная генерация уровней, система экипировки и грейдов, классы персонажа, навыки, несколько локаций с боссами.

## Запуск

Открыть папку проекта в **Godot 4.7** и запустить `scenes/Main.tscn`.

## Управление

| Клавиша | Действие |
|---|---|
| `WASD` / стрелки | Движение |
| `ЛКМ` | Атака |
| `E` | Взаимодействие (сундук, подбор предмета) |
| `I` | Инвентарь / экипировка |
| `O` | Характеристики и способности |
| `Tab` | Дерево навыков |
| `Esc` | Меню паузы / закрыть окно |

## Геймплей

### Локации
Каждый запуск начинается в **замке**. После убийства босса открывается случайная новая локация:

| Локация | Генерация | Враги |
|---|---|---|
| Замок | BSP-комнаты | Гоблин, Скелет-лучник |
| Лес | Открытое поле + домики | Гоблин, Шаман |
| Пещера | Клеточный автомат | Скелет, Имп |
| Болото | Открытое поле + водоёмы | Болотный голем |
| Руины | Крупные BSP-комнаты | Призрак, Лич |
| Вулкан | Пещера (плотные стены) | Магма-титан |

У каждой локации свой босс. После победы над ним уровень подземелья увеличивается и выбирается другая локация.

### Классы (выбор при старте)
| Класс | Бонус | Стартовое оружие |
|---|---|---|
| Воин | +1 Сила | Тренировочный меч |
| Лучник | +1 Ловкость | Самодельный лук |
| Маг | +1 Интеллект | Учебный жезл |

### Характеристики
- **Сила** — +2 урона в ближнем бою за очко
- **Ловкость** — +1.5 урона дальним / −2% КД атаки за очко
- **Живучесть** — +8 макс. HP за очко
- **Интеллект** — +3 лечение зельем / +5 мана / +1.5 урона жезлом за очко

За каждый уровень даётся 1 очко характеристик (окно `O`).

### Система грейдов
Экипировка имеет грейд, определяющий требование к уровню и редкость дропа:

| Грейд | Уровень | Шанс дропа |
|---|---|---|
| NG | 1+ | 100% базового |
| D | 5+ | 55% |
| C | 10+ | 30% |
| B | 20+ | 12% |
| A | 25+ | 5% |
| S | 30+ | 2% |

Надеть вещь выше своего уровня нельзя.

### Дроп
- Враги дропают вещи согласно индивидуальной таблице дропа (`data/enemies.json`)
- С одного врага или из одного сундука выпадает **не более 1 предмета каждого слота** (weapon, helmet и т.д.)
- Зелья выпадают без ограничений по количеству
- Стартовые оружия не выпадают никогда

### Навыки
При повышении уровня предлагается выбор из 3 случайных навыков (20 навыков, 4 уровня силы). Активные навыки видны в окне `O`.

## Структура проекта

```
data/           — JSON-конфиги (враги, предметы, навыки, грейды, лут)
scenes/         — сцены Godot (Main, Player, InventoryUI)
scripts/        — GDScript (логика игры, UI, генератор)
assets/
  sprites/      — спрайты персонажей, предметов, навыков
  tilesets/     — тайлы для каждой локации
tools/          — Python-скрипты генерации спрайтов
```

### Ключевые скрипты

| Файл | Назначение |
|---|---|
| `castle_generator.gd` | Процедурная генерация уровней, спавн врагов/боссов/сундуков |
| `player.gd` | Управление, атака, HP/мана, взаимодействие |
| `enemy.gd` | ИИ врагов, масштабирование по глубине, дроп лута |
| `inventory_ui.gd` | Экран инвентаря и экипировки |
| `stats_window.gd` | Окно характеристик и способностей |
| `class_select.gd` | Выбор класса при старте |
| `item_db.gd` | База предметов из items.json |
| `enemy_db.gd` | База врагов из enemies.json |
| `grade_db.gd` | Грейды экипировки из grade_config.json |
| `loot_config.gd` | Параметры лута и редкости |
| `skill_tree.gd` | Хранение и применение навыков |

## Настройка через файлы

Все ключевые параметры правятся в `data/` без изменения кода:

- `data/enemies.json` — статы врагов, таблицы дропа, спрайты
- `data/items.json` — все предметы, слоты, урон, грейд, флаг `no_drop`
- `data/player_stats.json` — стартовые характеристики и коэффициенты влияния
- `data/grade_config.json` — уровни требований и множители дропа грейдов
- `data/loot_config.json` — базовый шанс дропа, количество роллов в сундуке
- `data/rarity_config.json` — шанс редкой вещи и диапазон бонус-статов
- `data/skills.json` — все навыки, описания, эффекты
