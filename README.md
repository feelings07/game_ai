# Game AI

2D side-view RPG built with Godot 4 (GDScript).

## Структура проекта

- `scenes/` — сцены (Main, Player и др.)
- `scripts/` — GDScript-код
- `assets/sprites/` — спрайты персонажей и объектов
- `assets/tilesets/` — тайлсеты для уровней

## Запуск

Открыть папку проекта в Godot 4.3+ и запустить `scenes/Main.tscn`.

## Управление (заглушка)

- `A` / `←` — влево
- `D` / `→` — вправо
- `Space` — прыжок
